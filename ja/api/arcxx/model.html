arcxx::model

template<typename Derived>
struct model

table_name static member variable is required in model struct definition. arcxx::model is designed using CRTP(Curiously Recurring Template Pattern).

A Minimum model define code
struct ExampleTable : public arcxx::model<ExampleTable> {
    static constexpr auto table_name = "example_table";
};
