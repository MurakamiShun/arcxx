Find columns

Where clause

To find and get columns, use where().

auto connection = arcxx::sqlite3::connector::open("example.sqlite3");

auto sql_stmt = ExampleTable::where(ExampleTable::ID{1});

// print SQL code
std::cout << sql_stmt.to_sql() << std::endl;

// Execute data finding
const auto find_result = sql_stmt.exec(connection);

// decltype(error) == std::optional<arcxx::string>
if(!find_result){
    // error handling
    std::cout << "Error message:" << names_result.error() << std::endl;
}

// decltype(find_result) == tl::expected<std::vector<ExampleTable>, arcxx::string>
std::cout << "column count:" << find_result.value().size() << std::endl;
std::cout << "column name:" << find_result.value()[0].name << std::endl;

Where clause can chain the conditions. The conditions will join with AND. The below codes will be the same result.

ExampleTable::where(ExampleTable::ID::between(0,10)).where(ExampleTable::Name::like("user_"));

ExampleTable::where(ExampleTable::ID::between(0,10) && ExampleTable::Name::like("user_"));

Between clause

Between member function in attributes generates Between clause. The result type query_condition is used as argument of where().

ExampleTable::where(ExampleTable::ID::between(0,10));

Like clause

Like member function in string attribute generates LIKE clause. The result type query_condition is put into where clause.

ExampleTable::where(ExampleTable::Name::like("user_"));
